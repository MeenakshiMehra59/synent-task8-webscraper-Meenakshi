# Task 8: Web Scraper
Developed for the **Synent Technologies Python Development Internship**.

## Description
[cite_start]An advanced tool used to extract structured data from websites[cite: 128, 130].

## Features
- [cite_start]**Data Extraction:** Scrapes content using the `BeautifulSoup` and `requests` libraries[cite: 132].
- [cite_start]**Storage:** Saves the extracted data (titles/quotes) into a structured CSV file[cite: 136, 137].
- **Efficiency:** Handles HTTP requests and parses HTML content reliably.

## How to Run
1. Install dependencies: `pip install requests beautifulsoup4`
2. Run the command: `python web_scraper.py`
